# 北京天通苑小区房价历史分析报告 — 页面提纲与布局

> 本文档是 HTML 报告生成的**页面蓝图**，与《执行方案与 AI 交互流程》配套使用。
> 用途：指导 `report/build_report.py` 或 AI 直接生成 `report/天通苑房价分析报告.html`。
> 原则：**单文件、离线可打开、内联 SVG、数据可追溯**；视觉风格可与 demo 不同，但信息架构必须完整。

---

## 一、报告定位

| 维度 | 说明 |
|---|---|
| 报告类型 | 小区房价历史分析 + 政策事件归因 + 舆情联动 |
| 时间范围 | 近 3 年（默认 2023-07 ~ 2026-07，以 `config.TIME_RANGE` 为准） |
| 分析对象 | 北京 · 天通苑（整体均价曲线） |
| 叙事主线 | **宏观因果**（政策/事件 → 价格拐点） + **联动分析**（舆情情绪 ↔ 房价走势） |
| 技术形态 | 纯静态 HTML + CSS + 内联 SVG（**禁止**依赖外部 CDN / echarts / 图片外链） |

**核心洞察**：这不是纯数据展示页，而是「**价格走势讲清楚 → 事件解释拐点 → 舆情验证/补充叙事 → 给出审慎结论**」的分析报告。每一章都要回答一个问题，而不是堆图表。

## 二、整体页面结构（自上而下）

```
┌─────────────────────────────────────────┐
│  Cover 封面                              │
├─────────────────────────────────────────┤
│  01 数据概览                             │
│  02 三年均价走势（含 Top10 极值标注）      │
│  03 政策与关键事件时间线                  │
│  04 事件 × 价格叠加分析（宏观因果）        │
│  05 舆情热度与情绪联动                    │
│  06 极值成交与阶段性涨跌拆解              │
│  07 综合结论与展望                        │
│  08 数据来源与方法说明                    │
└─────────────────────────────────────────┘
```

**页数建议**：单页长滚动（与 demo 一致），8 个 section，每节 1~3 屏高度。

## 三、视觉与布局规范

### 3.1 风格方向（与 Manus demo 刻意区分）

| 元素 | Manus demo | 本报告建议 |
|---|---|---|
| 主色 | 深蓝 `#1F3A5F` + 米色底 | **墨绿 `#1A3C34` + 暖灰底 `#F5F3EF`**（偏「房产/金融简报」） |
| 涨跌色 | 红 `#B5462C` / 绿 `#5C7C4D` | 涨 `#2E7D57` / 跌 `#C0392B` / 平 `#8B8378` |
| 字体 | ui-sans-serif | 同上，标题可加 `Georgia` 斜体大号章节序号 |
| 封面 | 深色渐变全宽 | **左侧大标题 + 右侧迷你折线预览**（SVG 缩略） |
| 图表 | 手写 SVG | 继续内联 SVG，但折线图为主（非舆情条形图） |

### 3.2 通用布局组件

| 组件 | CSS 类名建议 | 用途 |
|---|---|---|
| 页面容器 | `.wrap` | `max-width: 1080px; margin: 0 auto; padding: 0 56px` |
| 章节 | `.section` | 顶部留 96px，`section-num` 大号水印序号 |
| 章节头 | `.section-head` | label + h2 + lead 导语 |
| 四格指标 | `.stat-row` / `.stat-cell` | 封面下 / 各节关键 KPI |
| 内容面板 | `.panel` | 图表、表格外框 |
| 引用块 | `.pull` | 核心结论 / 归因金句 |
| 趋势标签 | `.trend-tag.pos/.neg/.neu` | 涨/跌/持平 |
| 数据表格 | `table` | Top10、事件清单、样例舆情 |

### 3.3 图表规范（全部内联 SVG）

| 图表 | 所在章节 | 类型 | 必含元素 |
|---|---|---|---|
| 封面迷你走势 | Cover | 折线 sparkline | 36 个月点、期末涨跌色 |
| 三年均价曲线 | 02 | 折线 + 面积淡填充 | 月均价、YoY 辅助虚线、最高/最低 Top10 标注点 |
| 事件时间轴 | 03 | 横向时间轴 | 事件气泡、利好/利空颜色、日期刻度 |
| 事件叠加价格 | 04 | 双图层 | 下层折线（价格）+ 上层事件竖线/标记 |
| 舆情热度 | 05 | 柱状/折线 | 微博+小红书合并按月声量 |
| 情绪结构 | 05 | 堆叠面积或环形 | 正面/中性/负面占比 |
| 价格-情绪对照 | 05 | 双 Y 轴折线 | 左轴均价、右轴情绪指数 |
| 阶段涨跌柱 | 06 | 分组柱状 | 各阶段涨跌幅 % |

## 四、分节页面提纲（逐页说明）

---

### Cover 封面

**目的**：3 秒内让读者知道「分析谁、分析多久、核心结论倾向」。

**布局**：
```
┌──────────────────────────────────────────────────┐
│ [eyebrow]  小区房价专项分析                        │
│ [h1]  北京天通苑 · 三年房价历史分析                  │
│ [subtitle]  2023.07 — 2026.07 成交与政策事件复盘     │
│                                                    │
│ ┌─────────┬─────────┬─────────┬─────────┐          │
│ │ 当前均价 │ 3年涨跌  │ 最高成交 │ 最低成交 │          │
│ │ 元/㎡   │   %     │  元/㎡  │  元/㎡  │          │
│ └─────────┴─────────┴─────────┴─────────┘          │
│                              [右侧 SVG 迷你折线]     │
└──────────────────────────────────────────────────┘
```

**数据字段**（来自 `price_monthly.csv` + 分析结果）：
- `current_avg_price`：最新月均价
- `change_3y_pct`：3 年累计涨跌幅
- `peak_price` / `peak_date`：历史最高成交单价及日期
- `trough_price` / `trough_date`：历史最低成交单价及日期
- `sample_count`：成交样本条数
- `report_date`：报告生成日期

**文案示例**：
- eyebrow：`小区房价专项分析`
- subtitle：`基于链家/安居客成交与挂牌数据，结合政策事件与社媒舆情`

---

### Section 01 — 数据概览

**label**：`数据概览`  
**h2**：`天通苑三年房价全景摘要`  
**lead**：用 2~3 句话概括：整体趋势（阴跌/企稳/反弹）、波动幅度、与北京市/昌平区对比（如有）。

**布局顺序**：
1. **stat-row（4 格）**
   | 指标 | 说明 |
   |---|---|
   | 分析周期 | 36 个月 |
   | 成交样本 | N 条 |
   | 均价区间 | 最低~最高 元/㎡ |
   | 最大单月跌幅 | -X%（YYYY-MM） |

2. **正文段落**（150~250 字）：描述三年分阶段特征（如：2023 高位震荡 → 2024 阴跌 → 2025 筑底 → 2026 弱修复）。

3. **pull 引用块**（核心判断）：
   - headline：`三年主线：政策预期主导，本地配套提供下限支撑`
   - body：1 段话，点出 1~2 个最关键事件

**数据绑定**：`price_monthly.csv`、`attribution.py` 输出的 summary dict

---

### Section 02 — 三年均价走势

**label**：`价格走势`  
**h2**：`月度均价曲线与历史极值`  
**lead**：展示天通苑近 3 年月度均价变化，并标注历史成交价 **Top10 最高** 与 **Top10 最低** 节点。

**布局顺序**：
1. **主图 panel**：SVG 折线图（宽 900×高 320）
   - X 轴：月份（2023-07 … 2026-07）
   - Y 轴：元/㎡
   - 折线：月度均价
   - **红色高亮点 + 序号**：Top10 最高成交（1~10）
   - **绿色高亮点 + 序号**：Top10 最低成交（1~10）
   - 可选：安居客验证线（虚线，灰色）

2. **panel-caption**：数据来源、是否插值、双源差异说明

3. **h3 2.1** `关键价位节点`
   - 左右两列表格（或上下两个 table）：
     - 表 A：`Top10 最高成交` → 排名 | 日期 | 单价 | 总价 | 面积 | 来源
     - 表 B：`Top10 最低成交` → 同上

4. **h3 2.2** `同比 / 环比变化`
   - 小 SVG 柱状图：仅标注波动 >±3% 的月份
   - 或 compact table：月份 | 均价 | 环比 | 同比

**数据绑定**：
- `price_monthly.csv` → 折线
- `data/clean/top10_high.csv` / `top10_low.csv`（或由 analysis 输出）

---

### Section 03 — 政策与关键事件时间线

**label**：`事件时间线`  
**h2**：`三年影响房价的关键事件与文件`  
**lead**：梳理全国、北京市、天通苑本地三个层级中，对预期和成交产生显著影响的事件。

**布局顺序**：
1. **主图 panel**：SVG 横向时间轴（宽 900×高 280）
   - 三层泳道：`全国政策` / `北京楼市` / `天通苑本地`
   - 事件标记：圆点 + 短标签
   - 颜色：`利好` 绿 / `利空` 红 / `中性` 灰

2. **h3 3.1** `事件清单`
   - 完整 table（来自 `events.csv`）：

   | 日期 | 事件 | 层级 | 类型 | 方向 | 摘要 | 来源 |
   |---|---|---|---|---|---|---|

3. **h3 3.2** `事件分级说明`
   - 3~5 条 bullet：哪些是「预期型」、哪些是「落地型」

**必含事件（至少覆盖）**：
- 2023-09 北京「认房不认贷」
- 2023~2025 多轮 LPR / 房贷利率下调
- 2024 限购政策调整 / 普宅标准调整
- 公积金政策优化
- 地铁 17 号线开通/延伸（天通苑相关）
- 天通苑旧改 / 法拍房供给变化（如有数据）

**数据绑定**：`data/clean/events.csv`

---

### Section 04 — 事件 × 价格叠加分析（宏观因果）

**label**：`宏观归因`  
**h2**：`政策事件与价格拐点的对应关系`  
**lead**：将月度均价曲线与关键事件对齐，分析「事件窗口」前后的价格反应。**声明：相关不等于因果。**

**布局顺序**：
1. **主图 panel**：SVG 叠加图
   - 底层：均价折线（同 Section 02，可简化）
   - 上层：事件竖线（虚线）+ 事件名缩写
   - 拐点标注：▼ 局部高点、▲ 局部低点（由 attribution 算法给出）

2. **h3 4.1** `阶段划分`
   - 3~4 个 `.panel` 卡片，每卡片：
     - 阶段名（如「政策刺激窗口 2023Q3-Q4」）
     - 时间范围
     - 均价变化 %
     - 关联事件（2~3 条）
     - 1 段归因文字（100 字内）

3. **pull 块**（归因金句）：
   - `2023 下半年：认房不认贷释放置换需求，天通苑均价短暂企稳`
   - `2024 全年：预期走弱 + 挂牌增加，均价承压下行`

4. **h3 4.2** `归因矩阵 table`

   | 事件 | 事件后 1 月 | 事件后 3 月 | 价格反应 | 置信度 |
   |---|---|---|---|---|
   | 认房不认贷 | +0.X% | -0.X% | 短期企稳 | 中 |

**数据绑定**：`attribution.py` 输出的 `phases[]`、`event_impact[]`

---

### Section 05 — 舆情热度与情绪联动

**label**：`舆情联动`  
**h2**：`社媒声量、情绪与房价走势的对照`  
**lead**：基于微博、小红书关键词样本，观察「讨论热度 / 情绪倾向」是否与价格拐点同步。

**布局顺序**：
1. **stat-row（4 格）**
   | 指标 | 说明 |
   |---|---|
   | 舆情样本 | N 条（微博 X + 小红书 Y） |
   | 覆盖关键词 | 天通苑 / 北京楼市 / 认房不认贷 … |
   | 负面占比 | Z% |
   | 热度峰值月 | YYYY-MM |

2. **h3 5.1** `月度声量`
   - SVG 柱状图：按月帖子/笔记数

3. **h3 5.2** `情绪结构`
   - SVG 堆叠面积图 或 环形图：正面 / 中性 / 负面

4. **h3 5.3** `价格 - 情绪双轴对照`（**联动分析核心图**）
   - 左 Y：均价（折线）
   - 右 Y：情绪指数或负面占比（折线/柱）
   - 观察：价格下跌期是否伴随负面舆情上升

5. **h3 5.4** `代表舆情摘录`
   - `.hero-list` 6~10 条：
     - 平台 icon | 标题/摘要 | 日期 | 互动数 | 情感标签

**数据绑定**：`sentiment.csv`、原始 weibo/xhs json

**局限声明**（本 section 末尾一小段）：样本量、平台偏差、关键词召回率

---

### Section 06 — 极值成交与阶段性涨跌拆解

**label**：`结构拆解`  
**h2**：`高点、低点与阶段性涨跌来源`  
**lead**：从 Top10 极值与分阶段涨跌出发，解释「哪些月份/哪些因素」拉高了/压低了均价。

**布局顺序**：
1. **h3 6.1** `阶段性涨跌`
   - SVG 分组柱状：各阶段涨跌幅 %
   - 阶段示例：2023H2 / 2024H1 / 2024H2 / 2025 / 2026H1

2. **h3 6.2** `极值成交特征对比`
   - 两列 panel：
     - 高点 common：户型、面积段、是否学区/地铁口径
     - 低点 common：是否法拍、急售、特殊户型

3. **table**：阶段汇总

   | 阶段 | 起止 | 均价起 | 均价止 | 涨跌% | 主要解释 |
   |---|---|---|---|---|---|

**数据绑定**：Top10 明细 + `attribution.py` phases

### Section 07 — 综合结论与展望

**label**：`结论展望`  
**h2**：`综合判断与后续关注`  
**lead**：给出一个审慎、可辩护的结论，不做投资建议。

**布局顺序**：
1. **pull accent**（总判断，3~5 行）
2. **h3 7.1** `核心发现`（有序列表 4~6 条）
   - 例：三年累计跌幅 X%，波动主要来自政策预期而非单点事件
   - 例：2024 年舆情负面峰值与均价下行段高度重叠
   - 例：本地配套（地铁/旧改）对价格下限形成支撑
3. **h3 7.2** `后续 6~12 个月关注项`
   - 3~4 条：利率、限购、天通苑供给、轨交规划
4. **h3 7.3** `风险提示`
   - 样本偏差、因果局限、非投资建议

## 八、与 demo 的对照（便于面试讲解）

| 维度 | Manus 舆情报告 | 天通苑房价报告 |
|---|---|---|
| 核心曲线 | 七日传播量 | 三年月度均价 |
| 时间轴 | 舆情爆发阶段 | 政策/本地事件 |
| 情绪 | 负面话题分布 | 社媒情绪 vs 价格 |
| 极值 | 高峰互动笔记 Top N | 最高/最低成交 Top10 |
| 结论 | 舆情应对建议 | 房价归因 + 后续关注 |
| 图表类型 | 条形/环形/小时分布 | **折线/叠加/双轴** 为主 |

**一句话定位**：Manus 报告是「**事件 → 舆情传播**」；本报告是「**时间 → 价格 → 事件/舆情解释**」。
